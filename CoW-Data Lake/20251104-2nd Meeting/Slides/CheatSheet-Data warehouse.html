dashboard Application & BI Layer

Cloud-based business intelligence and analytics platforms

bar_chart
AWS QuickSight
Business Intelligence Service
Serverless BI service, pay-per-session pricing. Integrates seamlessly with Redshift, Athena, RDS, and other AWS data sources. ML-powered insights and natural language queries.
trending_up
Power BI Service
Cloud BI Platform (Microsoft)
Microsoft's cloud BI platform with deep integration with Azure Synapse, Office 365, and Teams. Real-time dashboards, AI-powered analytics, and enterprise governance.
analytics
Looker (Google Cloud)
Modern BI Platform
Now part of Google Cloud, native integration with BigQuery. Data modeling in LookML, embedded analytics capabilities, and version-controlled data definitions.

search Query Engine Layer (Data Warehouses)

Cloud data warehouses for fast SQL analytics

account_balance
Google BigQuery
Serverless Data Warehouse
Fully serverless, pay-per-query model. Separation of storage (Colossus) and compute (Dremel). Auto-scales to petabytes, built-in ML capabilities, and geospatial analysis.
waves
AWS Redshift
Managed MPP Cluster
Traditional cluster model (leader + compute nodes). You manage cluster size but get fine-grained control over cost and performance. Redshift Spectrum queries S3 data directly.
extension
Azure Synapse Analytics
Unified Analytics Platform
All-in-one: SQL warehouse + Spark pools + data pipelines in one workspace. Best for Microsoft ecosystem. Dedicated SQL pools (provisioned) or serverless SQL pools (on-demand).
ac_unit
Snowflake
Multi-Cloud Data Warehouse
Runs on AWS, Azure, or GCP. Separation of storage and compute with instant scaling. Pay-per-second billing, zero-copy cloning, time travel, and data sharing across organizations.
pageview
AWS Athena
Serverless Query Service
Query S3 data directly using SQL (based on Presto). Pay only for queries run. No infrastructure to manage. Great for ad-hoc analysis and data lake queries.

bolt Processing Layer (ETL & Data Processing)

Cloud services for data transformation and processing

flash_on
AWS EMR
Elastic MapReduce
Managed Hadoop/Spark clusters on EC2. Pay for compute by the hour. Auto-scaling, spot instances for cost savings. Great for batch processing and large-scale ETL jobs.
whatshot
Azure Synapse Spark Pools
Managed Apache Spark
Apache Spark within Synapse workspace. Jupyter notebooks, auto-pause/resume, tight integration with Synapse SQL and Pipelines. Pay only when running.
public
Google Cloud Dataproc
Managed Spark/Hadoop
Fast cluster provisioning (90 seconds), integrates with BigQuery and GCS. Pay-per-second billing. Auto-scaling and auto-deletion of idle clusters.
science
AWS Glue
Serverless ETL
Fully serverless Spark-based ETL. Auto-discovers schema with crawlers, generates PySpark code. Data Catalog for metadata. Pay only for job runtime.
sync
Azure Data Factory
Data Integration Service
Visual ETL/ELT pipelines, 90+ connectors, schedule and orchestrate workflows. Integrates with Synapse, Databricks, and Azure ML.
waves
Google Cloud Dataflow
Serverless Stream/Batch
Unified stream and batch processing based on Apache Beam. Fully managed, auto-scaling. Great for real-time data pipelines and streaming analytics.

storage Storage Layer (Object Storage & Data Lakes)

Cloud object storage for data lakes

inventory_2
AWS S3
Simple Storage Service
The gold standard for object storage. Unlimited capacity, 11 9's durability (99.999999999%). Storage classes (Standard, IA, Glacier), lifecycle policies, versioning, and replication.
water_drop
Azure Data Lake Storage Gen2
Hierarchical Storage
Built on Blob Storage with hierarchical namespaces for file system operations. Native integration with Synapse, Databricks, HDInsight. POSIX ACLs for fine-grained security.
cloud
Google Cloud Storage
Unified Object Storage
Multi-regional, strongly consistent object storage. Storage classes (Standard, Nearline, Coldline, Archive). Seamless integration with BigQuery for querying files directly.

settings_applications Orchestration Layer (Container Services)

Managed Kubernetes and container orchestration

settings_ethernet
AWS EKS
Elastic Kubernetes Service
Managed Kubernetes control plane. Auto-upgrades, auto-scaling with Karpenter or Cluster Autoscaler. Integrates with IAM, VPC, CloudWatch, and AWS services.
hexagon
Azure AKS
Azure Kubernetes Service
Free control plane (pay only for nodes). Integrated monitoring with Azure Monitor, autoscaling, Virtual Nodes for serverless bursting. Azure AD integration.
my_location
Google GKE
Google Kubernetes Engine
Most mature managed K8s (Google invented Kubernetes). Autopilot mode for zero-ops clusters. Binary Authorization, Workload Identity, and built-in security scanning.
sailing
AWS ECS/Fargate
Container Orchestration
AWS-native alternative to Kubernetes. ECS = managed container orchestration. Fargate = serverless containers (no EC2 instances to manage). Simpler than K8s.

computer Foundation Layer (Compute Infrastructure)

Cloud virtual machines and compute services

laptop
AWS EC2
Elastic Compute Cloud
Virtual servers on demand. 500+ instance types (general purpose, compute-optimized, memory-optimized, GPU, ARM). Spot instances for 90% discounts. Pay-per-second billing.
desktop_windows
Azure Virtual Machines
Windows & Linux VMs
Flexible VM sizes from B-series (burstable) to H-series (HPC). Spot instances, reserved instances (1-3 year commitments). Best Windows VM support (same company as Windows).
build
Google Compute Engine
VM Instances
Custom machine types (configure exact vCPU and RAM). Sustained use discounts (automatic). Live migration for maintenance (no downtime). Preemptible VMs for batch jobs.
